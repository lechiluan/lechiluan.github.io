<h2>Tại sao nên chọn Go (Golang)?</h2>
<p>
  Go được thiết kế bởi Google để giải quyết các vấn đề về hiệu suất và sự phức
  tạp của hệ thống lớn.
</p>

<h3>1. Concurrency tuyệt vời</h3>
<p>
  Với <code>Goroutines</code>, bạn có thể thực hiện hàng triệu tác vụ đồng thời
  mà không tốn nhiều tài nguyên như luồng (threads) truyền thống.
</p>

<pre><code class="language-go">package main
import "fmt"

func main() {
    go func() {
        fmt.Println("Hello từ Goroutine!")
    }()
    fmt.Scanln()
}</code></pre>

<h3>2. Compile nhanh và Static Binary</h3>
<p>
  Go biên dịch ra một file nhị phân duy nhất, giúp việc deploy lên Docker hay
  Kubernetes trở nên đơn giản và nhanh chóng.
</p>

<h3>3. Tính đơn giản</h3>
<p>
  Triết lý của Go là "Less is More". Ngôn ngữ này không có class, không có kế
  thừa phức tạp, giúp code dễ đọc và dễ bảo trì hơn.
</p>
