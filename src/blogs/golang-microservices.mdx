# Tại sao nên chọn Go (Golang)?

Go được thiết kế bởi Google để giải quyết các vấn đề về hiệu suất và sự phức tạp của hệ thống lớn.

## 1. Concurrency tuyệt vời

Với `Goroutines`, bạn có thể thực hiện hàng triệu tác vụ đồng thời mà không tốn nhiều tài nguyên như luồng (threads) truyền thống.

```go
package main
import "fmt"

func main() {
    go func() {
        fmt.Println("Hello từ Goroutine!")
    }()
    fmt.Scanln()
}
```

## 2. Compile nhanh và Static Binary

Go biên dịch ra một file nhị phân duy nhất, giúp việc deploy lên Docker hay Kubernetes trở nên đơn giản và nhanh chóng.

## 3. Tính đơn giản

Triết lý của Go là "Less is More". Ngôn ngữ này không có class, không có kế thừa phức tạp, giúp code dễ đọc và dễ bảo trì hơn.
