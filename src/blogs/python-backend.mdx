<h2>Sức mạnh của Python trong Backend</h2>
<p>
  Python không còn chậm chạp như trước. Với sự ra đời của các runtime như PyPy
  hay việc tối ưu hóa trong phiên bản 3.12+, Python đang trở lại mạnh mẽ.
</p>

<h3>FastAPI: Cuộc cách mạng Backend</h3>
<p>FastAPI đã thay đổi cách chúng ta viết API với Python:</p>
<ul>
  <li>
    <strong>Tốc độ:</strong> Ngang ngửa với Go và Node.js nhờ hỗ trợ{" "}
    <code>async/await</code>.
  </li>
  <li>
    <strong>Type Checking:</strong> Sử dụng Pydantic để đảm bảo dữ liệu luôn
    đúng.
  </li>
  <li>
    <strong>Auto Docs:</strong> Tự động tạo Swagger UI cực kỳ tiện lợi.
  </li>
</ul>

<pre><code class="language-python">from fastapi import FastAPI

app = FastAPI()

@app.get("/")
async def root():
    return {"message": "Hello World"}</code></pre>
